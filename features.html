✨ goldottlink의 강력한 기능

모든 OTT 링크를 한 곳에서 체계적으로 관리하세요

🎯 핵심 기능

📱

멀티 플랫폼 지원

Android와 iOS 모두에서 사용 가능한 크로스 플랫폼 앱입니다. Flutter 기반으로 개발되어 양쪽 플랫폼에서 동일한 사용자 경험을 제공합니다.

  • Android 6.0 이상
  • iOS 12.0 이상
  • 통일된 UI/UX
🗂️

계층적 폴더 구조

큰 주제 → 작은 주제 → 링크로 이어지는 3단계 계층 구조로 체계적으로 정리할 수 있습니다.

  • 무제한 폴더 생성
  • 드래그 앤 드롭 지원
  • 폴더별 색상 및 아이콘
  • 빠른 검색 및 필터링
🎨

다양한 뷰 모드

자신의 선호에 맞는 뷰 모드로 링크를 관리하세요.

  • 플림 모드: 카드형 스와이프 뷰로 시각적 탐색
  • 일반 모드: 리스트 형식으로 빠른 접근
  • 폴드 모드: 트리 구조로 계층적 탐색
🌐

링크 열기 모드

링크를 여는 방식을 선택할 수 있습니다.

  • 인앱 브라우저: 앱 내에서 바로 콘텐츠 시청
  • 팝업 모드: 간단한 팝업으로 빠른 확인
  • 외부 브라우저: 기본 브라우저로 열기

📊 데이터 관리

📁

Excel/CSV 임포트/익스포트

대량의 링크를 한 번에 가져오거나 내보낼 수 있습니다.

  • Excel (.xlsx) 파일 지원
  • CSV 파일 지원
  • 자동 폴더 생성
  • 데이터 검증 및 오류 처리
☁️

클라우드 동기화

Firebase 기반 클라우드 백업으로 여러 기기에서 데이터를 동기화하세요.

  • 자동 클라우드 백업
  • 여러 기기 간 동기화
  • 데이터 복원 기능
  • 백업 이력 관리
💾

로컬 백업

기기에 직접 백업 파일을 저장하고 복원할 수 있습니다.

  • JSON 형식 백업
  • 수동 백업 생성
  • 빠른 복원
  • 인터넷 연결 불필요

🔒 보안 기능

🔐

AES-256 암호화

군사급 암호화로 데이터를 안전하게 보호합니다.

  • 로컬 데이터 암호화
  • AES-256 알고리즘
  • 안전한 키 관리
  • 자동 암호화/복호화
👆

생체 인증

지문이나 얼굴 인식으로 앱을 보호하세요.

  • 지문 인식 지원
  • 얼굴 인식 지원 (Face ID)
  • 앱 잠금 기능
  • 빠르고 안전한 인증

🚀 공유 기능

📱

QR 코드 생성

링크를 QR 코드로 변환하여 쉽게 공유하세요.

  • 개별 링크 QR 코드
  • 폴더 QR 코드
  • 고해상도 이미지 생성
  • 카메라로 즉시 스캔
🔗

링크 공유

다양한 방법으로 링크를 공유할 수 있습니다.

  • 링크 복사
  • 소셜 미디어 공유
  • 이메일 공유
  • 메시지 공유

🔍 검색 및 필터

🔎

강력한 검색 기능

원하는 링크를 빠르게 찾을 수 있습니다.

  • 키워드 검색 (제목, 설명, URL)
  • 태그 기반 필터링
  • OTT 플랫폼별 필터
  • 날짜 범위 필터
  • 실시간 검색 결과
