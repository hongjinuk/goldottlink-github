### Usage
The language switcher automatically:
1. Creates a language toggle button in the navigation menu
2. Saves user's language preference to localStorage
3. Applies translations when the page loads
4. Updates all elements with data-i18n attributes
5. Supports both data-i18n (for text content) and data-i18n-placeholder (for input placeholders)

### Switching Languages
Users can switch between Korean (KO) and English (EN) by clicking the language toggle button (üåê) in the navigation menu.
