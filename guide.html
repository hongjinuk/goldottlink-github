📖 사용 가이드

goldottlink를 효과적으로 사용하는 방법을 알아보세요

1 앱 다운로드 및 설치

Google Play Store 또는 App Store에서 "goldottlink"를 검색하여 다운로드하세요.

  • Android: Google Play Store → "goldottlink" 검색 → 설치
  • iOS: App Store → "goldottlink" 검색 → 받기

2 계정 생성 또는 로컬 모드 선택

첫 실행 시 계정 생성 또는 로컬 모드 중 선택할 수 있습니다.

  • 계정 생성: 이메일과 비밀번호로 계정을 만들면 클라우드 동기화와 백업 기능을 이용할 수 있습니다
  • 로컬 모드: 회원가입 없이 기기에만 데이터를 저장합니다. 인터넷 연결이 필요 없습니다

3 링크 추가하기

OTT 링크를 앱에 추가하는 방법입니다.

  1. 메인 화면 우측 하단의 '+' 버튼 클릭
  2. 링크 URL을 입력하거나 붙여넣기 (OTT 플랫폼 자동 감지)
  3. 제목 입력 (선택사항, 자동으로 생성됨)
  4. 폴더 선택 (없으면 "미분류"에 저장)
  5. 태그 추가 (선택사항)
  6. 설명 입력 (선택사항)
  7. 링크 열기 모드 선택 (인앱/팝업/외부 브라우저)
  8. '저장' 버튼 클릭

4 폴더로 정리하기

링크를 체계적으로 정리하기 위해 폴더를 생성하세요.

  1. 하단 네비게이션에서 '폴더' 탭 선택
  2. '+' 버튼으로 새 폴더 생성
  3. 폴더명, 색상, 아이콘 선택
  4. 상위 폴더 선택 (하위 폴더 만들기)
  5. 링크를 폴더로 드래그하여 이동

5 뷰 모드 전환하기

자신에게 맞는 방식으로 링크를 볼 수 있습니다.

  • 플림 모드: 홈 화면에서 '플림 모드' 선택. 카드를 좌우로 스와이프하여 탐색
  • 일반 모드: 홈 화면에서 '일반 모드' 선택. 리스트로 보고 정렬 가능
  • 폴드 모드: 홈 화면에서 '폴드 모드' 선택. 폴더 구조로 탐색

6 검색 및 필터 사용하기

원하는 링크를 빠르게 찾는 방법입니다.

  1. 홈 화면 상단의 검색 아이콘 클릭
  2. 키워드 입력하여 실시간 검색
  3. '필터' 버튼으로 고급 필터 옵션 열기
  4. OTT 플랫폼, 태그, 날짜 범위로 필터링
  5. 활성 필터는 칩으로 표시되며 클릭하여 제거 가능

7 Excel/CSV로 가져오기

대량의 링크를 한 번에 가져오는 방법입니다.

  1. 설정 → 데이터 관리 → Excel/CSV 메뉴 선택
  2. '가져오기' 버튼 클릭
  3. Excel (.xlsx) 또는 CSV 파일 선택
  4. 자동으로 데이터 파싱 및 링크 생성
  5. 폴더가 없으면 자동으로 생성됨

파일 형식 예시:

  • 제목 | URL | OTT | 폴더 | 태그 | 설명 | 열기모드
  • 예: "킹덤 시즌1" | "https://netflix.com/..." | "Netflix" | "드라마/한국" | "좀비,역사" | "조선시대 좀비물" | "inapp"

8 백업 및 복원

데이터를 안전하게 보관하는 방법입니다.

로컬 백업:

  1. 설정 → 백업 & 복원 메뉴 선택
  2. '로컬 백업' 버튼 클릭
  3. 기기의 다운로드 폴더에 JSON 파일 저장
  4. 복원: 백업 파일 선택 → '복원' 버튼

클라우드 백업 (로그인 필요):

  1. 설정 → 백업 & 복원 메뉴 선택
  2. '클라우드 백업' 버튼 클릭
  3. Firebase Storage에 자동 업로드
  4. 다른 기기에서 로그인하여 복원 가능

9 QR 코드로 공유하기

링크를 다른 사람과 공유하는 방법입니다.

  1. 공유할 링크를 길게 누르기
  2. 'QR 코드 생성' 선택
  3. QR 코드 화면 표시
  4. 상대방이 카메라로 스캔하여 링크 확인
  5. 또는 '공유' 버튼으로 다른 앱으로 전송

10 보안 설정하기

데이터를 안전하게 보호하는 방법입니다.

생체 인증 설정:

  1. 설정 → 보안 메뉴 선택
  2. '생체 인증 사용' 토글 활성화
  3. 지문 또는 얼굴로 인증
  4. 앱 시작 시 인증 필요

데이터 암호화:

  1. 설정 → 보안 메뉴 선택
  2. '암호화 설정하기' 버튼 클릭
  3. AES-256 암호화 키 생성
  4. 로컬 데이터 자동 암호화
